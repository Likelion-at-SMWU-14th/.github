## About Us
[멋쟁이사자처럼 대학](https://likelion.university/)은 ‘내 아이디어를 내 손으로 실현한다’라는 가치 아래, 교육 프로그램을 통해 서비스를 개발하는 **웹 개발 동아리** 입니다. <br/>
현재, 총 80개의 대학이 함께하는 하는 국내 최대 규모 IT 창업 연합 동아리이며 그 중 **숙명여대도 14기에 함께**합니다. <br/>
